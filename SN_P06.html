Introducción

Como introducción describe el modelo difuso de inferencia de Tsukamoto. Puedes apoyarte de las referencias que consideres necesarias. Los datos del trabajo orignal son:
T. Tsukamoto. An approach to fuzzy reasoning method. In Madan M. Gupta. Ramohan K. Ragade, and Ronal R. Yager, editors, Advance in fuzzy set theory and applications, pages 137-149. North-Holland, Amsterdam, 1979
En los modelos de inferencia difusos tipo Tsukamoto, el consecuente de cada regla difusa si-enotnces está representado por un conjunto difuso con una función de membresía montónica, como se muestra en la figura. Como consecuencia, la salida inferida de cada regal se define como un valor certero inducido por el valor de la fuerza de disparo de la regla. El valor de salida certero es calculado de la misma manera que en los sistemas difusos sugeno. El model Tsukamoto no sigue estrictamente el proceso de inferencia composicional difusa.
*En este modelo la función consecuente es un set difuso con una función monotónica
*Cada regla tienen un output exacto.
*Este método no necesita defuzificacion, ya que agrega los outputs exactos de cada regla usando el promedio ponderado.
En general, para las aplicaciones de control se utilizan controladores de tipo Sugeno ya que son computacionalmente más eficientes, aunque el modelo de Mamdani también se utiliza por ofrecer los mejores resultados en cuanto a la calidad de respuesta del sistema a controlar, ya que representa de mejor forma las decisiones del experto.

Para el mecanismo de inferencia de Mamdani, el antecedente y la consecuencia son variables difusas. Sin embargo, para la inferencia de Sugeno, el antecedente es una variable difusa, pero la consecuencia es una constante o función lineal.
3.11 De la sección 2.2.1 FLC configuration, ¿Cuántas variables lingüisticas se tienen? ¿Que tipo de conjuntos difusos son los consecuentes? ¿Cuantas reglas si-entones se proponen? ¿Que sistemas de inferencia se utiliza? ¿Qué operaciones se utilizan para la Conjunción, Disyunción e Implicación?
Tenemos seis variables lingüisticas, 6 de entrada:
  1. Frecuencia cardiaca media
  2. Amplitud de la señal P
  3. Intervalo de duración de PR
  4. Duración de QRS complejo
  5. Intervalo de duración RR
  6. Intervalor de duración ST
y una de salida la cual es "Tipo de arritmia" , en total 7 variables ligüisticas.
Posteriormente, para cada entrada se proponen 3 conjuntos difusos:
  • Minimo (MIN)
  • Medio (MOY)
  • Máximo (MAX)
Dichos conjuntos tienen un universo de discurso distinto, el cual abarca desde el minimo valor posible hasta el máximo dependiendo de cada variable, además de que todos presentan una función de membresía tipo Gaussiana.
Las reglas si- entonces propuestas son 5
  • Si el ritmo cardiaco is MOY Y la amplitud P es MOY Y la duración del intervalo PR is MOY Y la duración del QRS complejo es MOY Y la duración del intervalo de ST is MOY entonces es un ritmo sinusal normal (NSR)
  • Si el ritmo cardiaco is MIN Y la amplitud P es MIN Y la duración del QRS complejo es MAX Y la duración del intervalo RR es MAX entonces es una contración ventricular prematura (PVC).
  • Si el ritmo cardiaco is MOY Y la duración del QRS complejo es MAX Y la duración del intervalo RR is MAX entonces son latidos de ritmo (P).
  • Si el ritmo cardiaco is MIN Y la amplitud P es MAX Y la duración del intervalo PR is MOY Y la duración del QRS complejo es MAX Y la duración del intervalo de ST is MAX entonces es un bloqueo en la rama derecha (RBBB).
  • Si el ritmo cardiaco is MIN Y la amplitud P es MIN Y la duración del intervalo PR is MOY Y la duración del QRS complejo es MAX entonces es un bloqueo en la rama izquierda (LBBB)
El sistema de inferencia utilizado es Mamdani cuyas salidas serán singletons.
Para la conjunción e implicación se realiza la operación de producto, para la disyunción se tiene la operación de suma.
